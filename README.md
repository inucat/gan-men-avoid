# gan-men-avoid

![Game screen](asset/screenshot.png)

Avoid flying faces.

## Build

```
javac cloud/inucat/GanMenAvoid/*.java
```

## Play

```
java cloud.inucat.GanMenAvoid.Entrypoint
```

## Control

- `UP`, `DOWN`, `LEFT`, `RIGHT` : Move
- `R` : Revive
- `SPACE` : Exit
